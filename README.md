### HOW TO RUN

#### installation:

```
npm install
```



then, you have to create a '.env' file inside server folder with these attributes:

```
JWT_SECRET;
GOOGLE_CLIENT;
MONGO_URI;
PORT;

```



#### NEXT, 

##### In server folder: 

```
npm run dev
```

##### in client folder

```
live-server --host:localhost
```
